# ESG Risk Assessment Dashboard

## Project Overview

This Power BI dashboard analyzes ESG risk indicators across multiple countries, helping identify climate readiness, adaptive capacity, carbon emissions, and renewable energy performance.

## Dashboard Features

- Average Readiness Index
- Average Adaptive Capacity Index
- Average CO₂ Emissions
- Average Renewable Energy Consumption
- CO₂ Emissions Treemap by Country
- Global ESG Risk Distribution Map
- Climate Risk vs Readiness Scatter Plot
- Year Slicer for interactive analysis

## Tools Used

- Power BI
- Power Query
- DAX
- Excel / CSV

## Files Included

- ESG Risk Assessment Dashboard.pbix
- ESG Risk Assessment Dashboard.png
- energy_economics_curated.csv

## Key Insights

- Compare ESG risk indicators across countries.
- Identify high-emission countries.
- Explore relationships between readiness and climate risk.
- Analyze renewable energy performance over time.
